My Projects

  • Etch Code

    My main programming project is Etch Code, a written version of Scratch. For example: 

    	Events.flagClicked:
		Looks.say("Etch Code")

    The idea is to provide an easy method to switch from Scratch, a visual language, to a "real" written language using a simplified intermediate step. It is currently alpha, and you can try it out at etchcode.appspot.com.

    In this project I have been learning about AngularJS, tokenizing and parsing a language, Github, and the basics of the Flask web framework.

    My friend Sam Schickler has been working on most of the backend parsing.

  • Chicken A Day

    Chicken A Day is a small toy project that provides subscribers with a daily chicken image pulled from Flickr. Subscribe at chickenaday.appspot.com (Why wouldn't you want a daily chicken photo?).

    In this project I learned about using Google App Engine and programmatic email sending through Mandrill.
